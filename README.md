# IoT LoRa Communication (Tx-Rx) with ESP Heltec

Proyek ini merupakan implementasi sistem **Internet of Things (IoT)** menggunakan dua perangkat ESP Heltec LoRa WiFi sebagai **transmitter (Tx)** dan **receiver (Rx)**. Data suhu dan kelembapan (sensor DHT/dummy) dikirim dari Tx ke Rx menggunakan komunikasi LoRa, kemudian diteruskan ke platform cloud ThingSpeak untuk monitoring secara real-time.

Selain itu, sistem juga mendukung **kontrol dua arah**, di mana perintah dari ThingSpeak dapat dikirim kembali ke Tx melalui Rx untuk mengaktifkan atau menonaktifkan perangkat. Pada pengembangan lanjutan, digunakan protokol MQTT untuk komunikasi data yang lebih ringan dan efisien.

Proyek ini mencakup integrasi komunikasi LoRa, cloud IoT, serta konsep monitoring dan kontrol jarak jauh dalam satu sistem yang sederhana namun fungsional.
